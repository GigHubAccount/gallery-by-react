## 环境搭建填坑
1、没有Gruntfile.js不是安装错误，而是generator-react-webpack V2.0 移除了Grunt（webpack替代）。可以执行 npm start 或者npm run serve 启动服务。
2、node运行后，直接把浏览器和git用ctrl+c、ctrl+d关闭，就相当于关闭网址端口与服务器。
3、npm start启动服务器。
